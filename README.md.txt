# JerseyStore

A web store for authentic football jerseys.

## Overview
JerseyStore is a responsive website where users can browse and purchase football jerseys from different teams around the world. Built with HTML, CSS, and JavaScript.

## Features
- Browse jerseys by team
- View product details
- Add to cart (frontend simulation)
- Responsive design for mobile and desktop
